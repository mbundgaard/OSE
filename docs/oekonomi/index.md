# Økonomi

## Oversigt

Økonomiafdelingen håndterer regnskab, fakturering og rapportering for OCC.

## Nuværende systemer

- **C5** - Regnskab (udfases)
- **Business Central** - Nyt regnskabssystem (implementering i gang)
- **Excel** - Samler data fra forskellige systemer til rapportering

## Nøglepersoner

- **Camilla** - Har været med i Mews-implementeringen på Hotel Odeon, "mega skarp" på systemskifte

## Kendte processer

*Afventer detaljeret kortlægning*

### Fakturering
- For messe: Økonomi sender fakturaer (via C5)
- For konference: Projektleder sender selv faktura (via Picasso)

### Rapportering
- Samler manuelt data fra forskellige systemer
- Konsoliderer i Excel

## Snitflader

- **Alle afdelinger** - Fakturering og rapportering
- **Messe** - Sender fakturaer på vegne af messe

## Kendte udfordringer

- "Voldsomt mange mennesker" sidder i økonomi pga. manuel dataindsamling
- Skal hente data fra mange forskellige systemer
- Når økonomi sender fakturaer for messe, kender de ikke arrangementet - kan ikke fange fejl

## Anbefaling fra mødet

Fakturering bør ske i det system der kender arrangementet (Opera), ikke i økonomi. Økonomi bør fokusere på bogføring, kontoafstemning og indbetalinger.
