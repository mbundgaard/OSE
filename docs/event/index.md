# Event

## Oversigt

Eventafdelingen håndterer events både internt i OCC og eksternt (f.eks. fester på slotte rundt på Fyn). Sidder sammen med Konference.

## Nuværende systemer

- **Picasso** - Booking og køresedler
- **Word** - Kontrakter

## Bemanding

Del af de ca. 8 medarbejdere der sidder sammen med Konference.

## Kendte processer

*Afventer detaljeret kortlægning*

### Interne events
- Events i OCC's lokaler
- Håndteres via Picasso ligesom konferencer

### Eksterne events
- Events uden for huset (slotte, lokationer på Fyn)
- Ligger under konferenceafdelingen
- Faktureres via Picasso

### Support til messe
- Leverer "pynt og gøgl" til messer
- Tilkobles når messe har brug for event-elementer

## Snitflader

- **Konference** - Deler kontor og systemer
- **Messe** - Leverer event-services
- **FB** - Forplejning til events

## Kendte udfordringer

- Samme som konference - uklarhed ved krydsende arrangementer
