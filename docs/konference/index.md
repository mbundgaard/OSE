# Konference

## Oversigt

Konferenceafdelingen sidder sammen med Event og håndterer booking og afvikling af konferencer i OCC's lokaler.

## Nuværende systemer

- **Picasso** - Booking af lokaler og ressourcer, køresedler
- **Word** - Kontrakter

## Bemanding

Ca. 8 medarbejdere (sidder sammen med Event).

## Kendte processer

*Afventer detaljeret kortlægning*

### Booking og tilbud
- Modtager forespørgsler
- Udarbejder tilbud
- Sender kontrakt via Word

### Planlægning
- Opretter køresedler i Picasso
- Koordinerer med FB omkring forplejning
- Koordinerer med Hotel Odense/Odeon omkring værelser

### Fakturering
- Projektleder sender selv faktura efter arrangement
- Fakturering sker via Picasso

## Snitflader

- **Messe** - Når arrangementer krydser (konference + messe)
- **Event** - Deler kontor, tæt samarbejde
- **FB** - Forplejning til arrangementer
- **Hotel Odense/Odeon** - Værelsesallokering
- **Økonomi** - Fakturering og rapportering

## Kendte udfordringer

- Når arrangementer involverer både konference og messe, er der uklarhed om ansvar (f.eks. hvem bestiller ekstra stole)
- Personale skal konsultere både køresedler (fra Picasso) og projektmanualer (fra Messe)
