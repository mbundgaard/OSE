# OSE - Odense Sport & Event

Vidensbank for OCC-projektet - dokumentation af arbejdsgange og processer.

## Om projektet

Dette repository indeholder dokumentation af arbejdsgange og processer for Odense Congress Center som grundlag for implementering af Oracle Opera Cloud (PMS) og Oracle Simphony (POS).
