## First Steps Checklist

Before beginning implementation, ensure you have:

- [ ] **Oracle Simphony compatibility** - Verify STS licensing and system version
- [ ] **Payment provider access** - Confirm merchant account credentials and API access
- [ ] **Hardware planning** - Review device requirements
- [ ] **Implementation timeline** - Plan deployment around restaurant operations
- [ ] **Technical contacts** - Identify IT support and Oracle Simphony administrators
- [ ] **Testing environment** - Prepare for demo mode testing and validation

!!! tip "Implementation Planning"
    Most implementations take 2-4 weeks from start to go-live, depending on complexity and testing requirements. Plan accordingly with your restaurant operations schedule.
