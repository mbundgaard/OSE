## Phase 1: Project Initiation (Week 1)
*Total: 1 day remote*

### Remote Activities
- Oracle provisions Simphony environment (10 business days)
- Initial system configuration
- Basic setup preparation

---

## Phase 2: Hardware Setup & Initial Training (Week 2-3)
*Total: 1 day on-site + 0.5 days remote*

### On-site Activities (1 day)
**Morning Session**
- Install test workstation at hotel
- Network configuration and connectivity testing
- Hardware setup and basic functionality testing
- Initial system walkthrough for super-users

**Afternoon Session**
- Introduction to EMC
- Properties, Revenue Centers, Menu items, Pages
- Introduction to Mobile Ordering concepts

### Remote Follow-up (0.5 days)
- Address any technical issues identified
- Remote configuration adjustments
- Prepare for next training phase

---

## Phase 3: Iterative Configuration & Training (Weeks 3-6)
*Total: 4 days remote across multiple sessions*

### Session 1: Menu & Pricing Setup (1 day)
- Menu configuration training
- Pricing structure setup
- Practice session assignment

### Session 2: Mobile Ordering Deployment (1.5 days)
- Mobile Ordering backend setup
- Device configurations
- Integration testing
- Payment integration setup

### Session 3: Revenue Centers & Reporting (1.5 days)
- Revenue center configuration
- Reporting structure setup
- Kitchen printing and order routing
- Department-specific configurations

---

## Phase 4: Event Configuration (Week 6)
*Total: 0.5 days remote*

### Remote Configuration Session (0.5 day)
- Add event-specific revenue centers
- Configure mobile devices for event operations
- Set up reporting separation (restaurant vs events)
- Test event order workflows

### Client Activities
- Test event operations in parallel with existing EventPOS
- Validate reporting accuracy
- Confirm mobile device functionality for events

---

## Phase 5: Parallel Testing & Training (Weeks 7-8)
*Total: 1 day remote*

### Remote Activities (1 day)
**Training Sessions** (4 hours total - multiple calls)
- Restaurant staff training
- Event staff training
- Management reporting training

**System Testing Support**
- Parallel testing with existing systems
- Data validation and reporting verification
- Performance testing and optimization
- Issue resolution

### Client Activities
- Full staff familiarization with new system
- Parallel operations testing
- Process refinement
- Final configuration requests

---

## Phase 6: Go-Live Support (Week 8-9)
*Total: 1.5 days on-site*

### Pre Go-Live (0.5 days on-site)
- Final system verification
- System reset for go-live
- Hardware final checks
- Team readiness assessment

### Go-Live Day (1 day on-site)
**Morning (4 hours)**
- System activation
- Staff support during initial operations
- Real-time issue resolution
- Process monitoring

**Afternoon (4 hours)**
- Continued operational support
- Performance monitoring
- Staff confidence building
- Documentation of any issues
